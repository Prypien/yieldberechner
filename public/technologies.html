VM Yield Modelle

Formeln und Hinweise (read-only).

Poisson (VM)

Formel: VM = exp(-D * A)

Beschreibung: Poisson-Modell. Verwendet keine zusätzlichen Parameter.

Murphy (VM)

Formel: VM = (1 - exp(-D * A)) / (D * A)

Beschreibung: Glättet Poisson bei kleinen Flächen.

Seeds (VM)

Formel: VM = exp(-sqrt(D * A))

Beschreibung: Schwächere Flächenabhängigkeit.

Bose–Einstein (VM)

Formel: VM = (1 / (1 + D * A)) ^ n

Beschreibung: Exponent n steuert Cluster-Effekt.

Negative Binomial (VM)

Formel: VM = (1 + (D * A) / alpha) ^ (-alpha)

Beschreibung: alpha steuert Defekt-Clusterung.
